# Hands-On-Data-Analysis-and-ML

A collection of **hands-on projects** focused on real-world data analysis, Exploratory Data Analysis (EDA), data cleaning, feature engineering and basic Machine Learning models.

### About This Repository
This repository showcases my practical journey in Data Science and Machine Learning. 

Each project starts with real-life datasets. I perform end-to-end analysis including:
- Data loading and understanding
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) with visualizations
- Feature engineering
- Building basic Machine Learning models (where applicable)
- Insights and conclusions

Most projects combine **strong data analysis + ML modeling**, while some focus purely on in-depth EDA.

### Technologies & Tools Used
- **Python**
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Scikit-learn
- Jupyter Notebooks
